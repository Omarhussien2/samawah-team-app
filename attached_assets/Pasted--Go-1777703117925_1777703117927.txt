المهم جدًا:
- ممنوع تشغيل التطبيق أو استضافته على سيرفرات Replit/Rebuilt أو أي مساحة داخلية للمنصة.
- المنصة تستخدم فقط للبناء وتوليد الكود.
- يجب إنشاء Repository على GitHub ورفع كل الكود عليه.
- يجب ربط المشروع بـ Vercel للنشر الخارجي.
- يجب أن يعمل التطبيق من Vercel فقط.
- قاعدة البيانات تكون على Supabase.
- أي Secrets أو Environment Variables تكون في Vercel وSupabase فقط، وليس داخل Replit/Rebuilt.
- لا تستخدم SQLite أو تخزين محلي دائم داخل منصة البناء.
- لا تعتمد على ملفات Excel/CSV كقاعدة بيانات نهائية. يتم استخدامها فقط للاستيراد الأولي.

الهدف:
بناء Web App مجاني/منخفض التكلفة لفريق صغير تقريبًا 6 أشخاص، يساعدهم على:
1. إدارة المشاريع.
2. إدارة المهام عبر Kanban Board.
3. تقليل إدخال البيانات اليدوي.
4. عرض لوحة تحكم واضحة.
5. إرسال تنبيهات وتذكيرات آلية.
6. تسهيل تحديث حالة المهام من أعضاء الفريق.
7. دعم اللغة العربية بالكامل RTL.
8. تصميم حديث قريب من Asana من حيث البساطة والنظافة، وليس نسخة حرفية.

الجداول المطلوبة في Supabase:

1. profiles
- id uuid primary key references auth.users(id)
- full_name text
- email text unique
- avatar_url text
- role text check in ('admin','project_manager','member')
- active boolean default true
- created_at timestamptz default now()

2. projects
- id uuid primary key default gen_random_uuid()
- legacy_project_id text unique
- name text not null
- manager_id uuid references profiles(id)
- manager_name text
- path text
- current_stage text
- status text default 'active'
- start_date date
- end_date date
- total_budget numeric default 0
- description text
- logo_url text
- progress numeric default 0
- created_at timestamptz default now()
- updated_at timestamptz default now()

3. project_members
- id uuid primary key default gen_random_uuid()
- project_id uuid references projects(id) on delete cascade
- user_id uuid references profiles(id) on delete cascade
- role_in_project text default 'member'
- created_at timestamptz default now()

4. tasks
- id uuid primary key default gen_random_uuid()
- legacy_task_id text unique
- project_id uuid references projects(id) on delete cascade
- title text not null
- sub_task text
- category text
- owner_id uuid references profiles(id)
- owner_name text
- status text
- board_column text
- priority text default 'medium'
- start_date date
- due_date date
- cost numeric
- quantity_total numeric
- quantity_done numeric
- progress numeric default 0
- schedule_status text
- alert_level text
- alert_message text
- alert_action text
- days_to_due integer
- sort_order integer default 0
- created_at timestamptz default now()
- updated_at timestamptz default now()

5. challenges
- id uuid primary key default gen_random_uuid()
- project_id uuid references projects(id) on delete cascade
- task_id uuid references tasks(id) on delete set null
- title text not null
- description text
- status text default 'open'
- owner_id uuid references profiles(id)
- risk_impact text
- risk_type text
- resolution text
- created_at timestamptz default now()
- updated_at timestamptz default now()

6. documents
- id uuid primary key default gen_random_uuid()
- project_id uuid references projects(id) on delete cascade
- task_id uuid references tasks(id) on delete set null
- title text not null
- url text
- file_path text
- type text
- created_by uuid references profiles(id)
- created_at timestamptz default now()

7. comments
- id uuid primary key default gen_random_uuid()
- task_id uuid references tasks(id) on delete cascade
- user_id uuid references profiles(id)
- body text not null
- created_at timestamptz default now()

8. notifications
- id uuid primary key default gen_random_uuid()
- user_id uuid references profiles(id)
- project_id uuid references projects(id) on delete cascade
- task_id uuid references tasks(id) on delete cascade
- type text
- title text
- body text
- sent_via text
- sent_at timestamptz
- read_at timestamptz
- created_at timestamptz default now()

9. automation_logs
- id uuid primary key default gen_random_uuid()
- type text
- status text
- payload jsonb
- error text
- created_at timestamptz default now()

10. project_templates
- id uuid primary key default gen_random_uuid()
- name text not null
- description text
- category text
- created_at timestamptz default now()

11. task_templates
- id uuid primary key default gen_random_uuid()
- template_id uuid references project_templates(id) on delete cascade
- title text not null
- sub_task text
- category text
- default_duration_days integer
- default_priority text default 'medium'
- sort_order integer default 0

  - lib/notifications/templates.ts

الأمان:
- Supabase Row Level Security يجب أن يكون مفعّلًا.
- Admin يرى كل شيء.
- Project Manager يرى مشاريعه ومهامها.
- Member يرى المشاريع التي هو عضو فيها ومهامه.
- لا تعرض Service Role Key في الواجهة.
- Service Role يستخدم فقط في API routes الآمنة وCron.
- تحقق من CRON_SECRET في كل cron endpoint.
- لا تضع أي secrets في GitHub.
- أضف .env.example فقط.
